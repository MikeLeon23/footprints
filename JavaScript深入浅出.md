# 第4章 对象

## 4-1 对象概述

> 对象是一个包含相关数据和方法的集合（通常由一些变量和函数组成，我们称之为对象里面的属性和方法。

对象中包含一系列属性，这些属性是**无序**的。每个属性都有一个**字符串**key和对应的value。

### JS数据类型

- 原始类型

  - string
  - boolean
  - null
  - undefined

- 对象

  - Function

  - Array

  - Date
